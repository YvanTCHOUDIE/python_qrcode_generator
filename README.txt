Introduction
	This Python QR Code Generator is a Python-based application that allows users to easily generate QR codes from any text or URL. 
	This tool can be used for a variety of purposes, including linking to websites, providing quick access to contact information, or creating promotional material.
	With this library, users can create high-quality, scannable QR codes in a matter of seconds.
	The sample lines of codes here allow to generate a set of qr codes based on an input csv file having data: the csv data allow to determine the http links to be encrypted into qr codes
Features
	Easy to Use: Simple command-line interface (CLI) for quick QR code generation.
	Text & URL Support: Can generate QR codes for both text and URLs.
		constructing the http link based on csv input file
		generate the qr codes for each http link
		store the qr codes as images into a local folder
Installation
	Prerequisites
		Python 3.6+ (recommended)
		pip (Python package manager)
	Install qrcode library (python -m pip install qrcode)
	Clone / Download the project and Navigate to the project folder
Usage
	python qr_code_generator.py
